# Project Backup Manager

VRChatプロジェクト向け自動バックアップ・世代管理・復元ツール。
Unityの予期せぬクラッシュによるデータ紛失を防止します。

## 主な機能

- **自動バックアップ** — 指定間隔で Assets/ 配下とシーンを自動保存
- **手動バックアップ** — ボタン一つで即座にバックアップを取得
- **世代管理** — 最大世代数を超えた古いバックアップは自動削除
- **重複排除** — コンテンツアドレッサブルストアにより同一ファイルは1回だけ保存
- **選択復元** — 特定のファイルだけを任意の世代から復元
- **全体復元** — Assets/ 全体をバックアップ時の状態に書き戻し
- **完全ロールバック** — バックアップ後に追加されたファイルの検出・削除まで対応
- **シーン自動保存** — 未保存シーン（Untitled）を自動的に保存してからバックアップ
- **バックグラウンド実行** — バックアップ処理がエディタ操作をブロックしない

## インストール

### 方法 1: unitypackage からインポート（Booth）

1. [Booth](https://mutoulab.booth.pm/) から最新の `.unitypackage` をダウンロード
2. Unity プロジェクトを開いた状態で `.unitypackage` をダブルクリック
3. Import ダイアログで「Import」をクリック
4. `Packages/com.mutoulab.project-backup-manager` にインストールされます

### 方法 2: UPM（Git URL）

1. Unity メニュー → `Window` → `Package Manager`
2. 左上の `+` → `Add package from git URL...`
3. 以下のURLを入力:

```
https://github.com/MutouLab/ProjectBackupManager.git
```

バージョンを固定する場合は末尾にタグを指定:

```
https://github.com/MutouLab/ProjectBackupManager.git#v0.1.0
```

### 方法 3: VCC（VRChat Creator Companion）

1. VCC を開き、`Settings` → `Packages` → `Add Repository`
2. 以下のURLを入力:

```
https://mutoulab.github.io/ProjectBackupManager/index.json
```

3. プロジェクトの `Manage Project` から `Project Backup Manager` を追加

## 使い方

### ウィンドウを開く

Unity メニュー → `MutouLab` → `Project Backup Manager`

### 画面構成

ウィンドウは2カラムレイアウトです。

**左カラム**
- バックアップボタン + ステータスバー（前回/次回/ストア容量）
- 3つのタブ:
  - **バックアップ一覧** — 保存済みの世代を時系列で表示
  - **詳細設定** — 自動バックアップの間隔・世代数・保存先の設定
  - **使い方** — 操作ガイド

**右カラム**
- 復元パネル — 世代を選択するとファイル一覧が表示され、復元操作が可能

### バックアップを取る

- **手動**: 「今すぐバックアップ」ボタンをクリック
- **自動**: 詳細設定タブで有効化（デフォルト: 有効 / 10分間隔）

バックアップはバックグラウンドで実行されるため、作業を中断する必要はありません。

### 復元する

1. バックアップ一覧から復元したい世代をクリック
2. 右カラムにファイル一覧が表示されます
3. 復元方法を選択:

| ボタン | 動作 |
|--------|------|
| **選択を復元** | チェックを入れたファイルのみ復元 |
| **全て復元** | バックアップ内の全ファイルを復元（追加ファイルは残る） |
| **完全ロールバック** | 全ファイル復元 + バックアップ後に追加されたファイルを検出・削除 |

復元後、シーンは自動的にリロードされます。

### 保存先を変更する

1. 詳細設定タブを開く
2. 保存先セクションの「保存先を変更...」ボタンをクリック
3. エクスプローラーで新しいフォルダを選択
4. 既存データの移動を確認するダイアログが表示されます

## 設定項目

| 項目 | デフォルト | 説明 |
|------|-----------|------|
| 自動バックアップ | 有効 | 指定間隔での自動バックアップの有効/無効 |
| 間隔（分） | 10 | 自動バックアップの実行間隔 |
| 最大世代数 | 12 | 保持するバックアップの最大数 |
| 保存先 | BackupStore | バックアップデータの保存フォルダ（プロジェクトルート相対） |

設定は `EditorPrefs` に保存され、プロジェクトごとに独立しています。

## データ構造

```
ProjectRoot/
├── Assets/
├── Packages/
│   └── com.mutoulab.project-backup-manager/   ← ツール本体
└── BackupStore/                              ← バックアップデータ
    ├── content/                              ← SHA-256ハッシュ名のファイル（重複排除）
    └── generations/                          ← 世代マニフェスト（JSON）
```

- `content/` にはファイルの内容がSHA-256ハッシュをファイル名として保存されます
- 同じ内容のファイルは複数の世代で共有されるため、ストレージ効率が高くなります
- `generations/` には各世代のファイルリスト・タイムスタンプ・メタデータがJSON形式で保存されます

## 既存プロジェクトへの導入

既存のプロジェクトにも安全に導入できます:

- `.unity` シーンファイルが既に存在する場合 → そのファイル名を継承
- `.unity` が存在しない場合 → プロジェクト名でシーンを自動保存

## アンインストール

1. Unity メニュー → `Window` → `Package Manager`
2. `Project Backup Manager` を選択
3. `Remove` をクリック
4. 必要に応じて `BackupStore/` フォルダを手動削除
